<h2>About 훈민정음</h2>

사용자는 **초성 퀴즈, 유의어 맞추기, 놀이 제작** 등을 통해 **현대인의 어휘학습능력**을 기르고, 한국어와 그 역사적 의미를 자연스럽게 배울 수 있습니다.
게임의 본질을 교육적 목적에 충실하게 맞추어 각 단계에서의 도전과 성취는 사용자가 단순히 문제를 푸는 것을 넘어, 한글에 대한 흥미와 애정을 키울 수 있도록 설계되었습니다.

<br>

## 주요 기능
- **놀이 선택**
  - 초성 퀴즈, 유의어 퀴즈, 놀이 제작 기능 중 원하는 모드를 선택해 게임을 시작
- **초성 퀴즈**
  - 타이머와 힌트 시스템으로 긴장감과 몰입감을 제공
  - 퀴즈 DB에서 무작위로 초성을 출제하며, 3회 오답 시 정답을 제공
    게임 종료 후 빠른 시간·높은 정확도 기준으로 랭킹에 자동 등록
- **유의어 퀴즈 & 랭킹 시스템**
  - 3개의 단어 중 유의어를 선택해 언어적 사고력을 강화
  - 점수 차감 방식으로 경쟁 요소를 부여해 학습의 흥미를 높임
  - 결과는 랭킹 시스템을 통해 실시간 비교
- **놀이 제작 기능**
  - 사용자가 초성·유의어 퀴즈 문제를 직접 생성하고 공유
  - 직관적인 UI로 문제 등록·삭제가 가능하며, 생성된 문제는 실제 게임 내에서 랜덤으로 출제되어 다른 사용자가 풀기 가능
- **학습 위젯**
  - 국립국어원 API를 통해 단어의 뜻과 예문을 실시간 제공
  - 일일 단어 퀴즈와 수능 국어 어휘 기반 단어 노출로 어휘력 향상
  - 하루 두 번 자동 업데이트되어 적절한 학습 빈도를 유지
